## DETECÇÃO DE HTML (REGRA DETERMINÍSTICA)

Considere que há HTML SOMENTE se pelo menos uma das condições abaixo for atendida:

- Existirem tags completas no formato: <tag>...</tag>
- Existirem strings contendo "<tag" seguido de ">"
- Houver concatenação de strings que claramente forma markup HTML

IGNORE completamente:

- Operadores (<, >)
- Generics (ex: List<string>)
- Comparações lógicas
- XML incompleto ou não relacionado a HTML
